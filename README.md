\# 📊 Portfolio Optimization Tool



This project is a Streamlit-based application that helps users optimize a portfolio of stocks using modern portfolio theory.



\## Features

\- Input stock tickers

\- Adjust risk-free rate

\- Set max allocation constraints

\- Generate optimal portfolio weights

\- Visualize efficient frontier



\## Technologies Used

\- Python

\- Streamlit

\- pandas

\- NumPy

\- yfinance

\- SciPy

\- Matplotlib



\## How to Run

pip install -r Requirements.txt

streamlit run APP.py
